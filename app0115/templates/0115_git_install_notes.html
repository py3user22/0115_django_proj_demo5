~~#0115 basic git cmds for install
------------------------------
clone a remote url on a new directory
____#note: will create a .init for directory so can leave not init

>> git clone https://github.com/py3user22/0115_django_template_demo1.git main


change the branch to another
---------------------
>> git branch -M main

>> git push -u origin main


change remote url to another
----------------------
>> git remote set-url origin https://github.com/py3user22/0116_django_template_demo6.git


check what remote url have set in local for push/pull
>> git remote -v


change to another branch
--------------------------
>> git switch master
